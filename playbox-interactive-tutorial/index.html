Playbox Tutorial

Your TV arcade, from power-on to first ROM.

A hands-on walkthrough for the dedicated retro gaming box: boot into the library, play with the controller, add games, connect from phone or laptop, and fix the usual gremlins.

1. What you\'ve got

Playbox is a dedicated retro console on a small PC. It boots into ES-DE, plays ROMs through RetroArch (and a few standalone emulators), and exposes web tools for library management.

On the TV

  • ES-DE game browser
  • Controller-first navigation
  • Save states, fast-forward, quit hotkeys

On the network

  • RomM — drag-and-drop ROM manager
  • Syncthing — file sync
  • VNC — see the live TV screen remotely
  • SSH — admin shell when needed
This public tutorial deliberately omits passwords and private admin secrets. Use your private vault notes for credentials.

2. Power on and play

No login screen. Press power, wait for ES-DE, grab the 8BitDo, go.

First session

  1. Press the power button
  2. Wait for the game library (ES-DE)
  3. D-pad to browse systems / games
  4. Press A to launch
  5. Hold Select + Start to quit back to the menu

In-game shortcuts

3. Controller map

Default pad is an 8BitDo SN30 Pro in Switch/USB mode. Labels are meant to match what games expect.

Bluetooth pairing can show connected without creating a working input device. If buttons die after a reboot, replug USB and restart the kiosk service.

4. Add games (ROMs)

Three ways in. RomM is the easy path for most people.

Easiest RomM web UI

  1. Open RomM in a browser
  2. Library → Upload ROMs
  3. Drag and drop files
  4. Run Scan for metadata/covers
  5. In ES-DE: Start → scan for new games

Other paths

  • Syncthing — drop files in the shared folder; they land under /srv/roms/
  • SCP/SSH — copy straight into the platform folder, then scan
scp game.zip user@PLAYBOX:/srv/roms/snes/

7. Make it fancy

Optional polish once the basics work.

CRT shaders

If menu access is unlocked for maintenance: open RetroArch shaders and try crt-easymode or crt-royale, then save a core preset.

Saves

  • Save state — instant snapshot, even when the game has no save feature
  • In-game save — normal RPG/battery saves

Themes

ES-DE → Start → UI Settings → Theme Set. Try a modern theme, then browse.

Achievements

RetroAchievements can be enabled in RetroArch settings if you want modern trophy energy on old games.

8. Fix common issues

Most failures are kiosk, controller, Docker, or display related.

Black screen / ES-DE dead

sudo systemctl status arcade-kiosk
sudo systemctl restart unbind-fbcon
sudo systemctl restart arcade-kiosk

Controller silent

ls /dev/input/js*
groups arcade
# replug USB, then:
sudo systemctl restart arcade-kiosk

RomM / Syncthing down

cd /srv/apps
docker compose ps
docker compose logs
sudo systemctl restart playbox-stack

Game launches to black

  • Try another core for that system
  • Confirm ROM extension/folder match
  • PS1 needs matching .cue + .bin
TV black bars are often underscan on the Samsung side: Picture Size → Screen Fit / Just Scan.

9. Admin quick commands

For the person who maintains the box. No secrets included.

htop

# restart game frontend
sudo systemctl restart arcade-kiosk

sudo systemctl restart playbox-stack

# logs

journalctl -u playbox-stack -f

# safe reboot
sudo reboot
